Manual test flows

  • 1. One-off purchase
    • ☐ Go to /captions → checkout (one-off) → Stripe (4242...) → thank-you page
    • ☐ Intake link appears within ~5–10 sec → Fill out form → Submit
    • ☐ Email: confirmation + intake link, then captions PDF within ~5–15 min
  • 2. Subscription purchase
    • ☐ Go to /captions → subscription → Stripe → thank-you → intake
    • ☐ Submit form → Create account (if prompted)
    • ☐ Check email for captions + stories
  • 3. Logged-in account
    • ☐ Log in → /account → see packs, Manage billing, Edit form
    • ☐ Download captions PDF
  • 4. Billing auth
    • ☐ Without login: try change subscription from intake → error/redirect
    • ☐ With login: change subscription → succeeds
  • 5. Intake link by email
    • ☐ Without login: thank-you fallback "enter email" → "Please log in"
    • ☐ With login: matching email → intake link
  • 6. Data export
    • ☐ Log in → visit /api/auth/export-data → JSON with account + orders
  • 7. Error pages
    • ☐ /nonexistent-page → 404

Quality checks (when reviewing PDFs)

  • ☐ Captions: Business-specific language, not generic "founder/strategy"
  • ☐ Key date: Launch day content uses correct month (e.g. April not May)
  • ☐ Stories: Same date phasing — teasers before, announce on day, thank-you after
  • ☐ Brand name: Real business name used, not substituted
  • ☐ Variety: No repetitive openers or phrases across days
  • ☐ Completeness: All 30 days present, no empty blocks
