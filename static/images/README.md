# Logo and images

**Hero logo:** Add your logo file as `logo.png` (or `logo.svg`) here.

The landing page hero uses `logo.png`. To use SVG instead, edit `templates/landing.html` and change `images/logo.png` to `images/logo.svg`.

**Recommended (to avoid pixelation when large):**
- **PNG:** Transparent background, **at least 1000â€“1200px wide** for crisp hero display
- **SVG:** Best for crisp scaling at any size
- **Color match:** Export with sRGB profile so #000000 in the image matches the site

If the file is missing, the hero falls back to the "Lumo 22" text.
