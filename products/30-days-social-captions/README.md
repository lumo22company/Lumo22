# 30 Days of Social Media Captions — Product Package

A sellable, done-for-you caption system for professionals and founders. Tone: confident, editorial, modern, premium. No hype, no emojis, no generic AI language.

---

## Files in this package

| File | Use |
|------|-----|
| **STEPS_TO_SELL_30_DAYS_CAPTIONS.md** | Step-by-step checklist to get the product on your website and selling: Stripe, product page, intake, delivery. Start here. |
| **AUTOMATION_SETUP.md** | How to turn on the full automated flow: Stripe webhook, DB table, intake → AI generation → delivery email. |
| **SETUP_STEP_BY_STEP_ASSUME_NOTHING.md** | Start here if you're new: step-by-step setup from zero (database, Stripe, webhook, .env, test). |
| **PRODUCT_30_DAYS_SOCIAL_CAPTIONS.md** | Master document: full framework, intake, 5-category structure, caption templates, output format, and product description. Use this to deliver the service. |
| **INTAKE_QUESTIONNAIRE.md** | Client-facing intake form (text). The live form is at `/captions-intake` on the site. |
| **SAMPLE_DELIVERY_FORMAT.md** | Example of how a delivered month looks (Days 1–5 filled in). Use as the template for client deliveries. |
| **PRODUCT_COPY_WEBSITE.md** | Website-ready copy: headline, summary, what’s included, who it’s for, problem solved, CTAs. Copy-paste onto sales or product pages. |

---

## Quick start

1. Put **PRODUCT_COPY_WEBSITE.md** content on your sales or product page.
2. Send **INTAKE_QUESTIONNAIRE.md** (or equivalent form) to each client before writing.
3. Write 30 captions using the framework and templates in **PRODUCT_30_DAYS_SOCIAL_CAPTIONS.md**.
4. Deliver using the structure in **SAMPLE_DELIVERY_FORMAT.md** (one .md or PDF per client per month).

---

*Product package complete. Ready to sell and deliver.*
